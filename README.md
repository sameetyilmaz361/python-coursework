# python-coursework
A collection of exercises and coursework from university classes and online courses, tracking my progression in Python for data analysis and scientific computing.

## Contents

- **`bme268/`** — Exercises from BME268 (university course): Python fundamentals, NumPy, Matplotlib, and SciPy.
- **`edx/`** — Exercises from HarvardX's "Using Python for Research": NumPy, pandas, statistical analysis, and machine learning basics (k-Nearest Neighbors).

## Tech stack

Python, NumPy, pandas, Matplotlib, SciPy, scikit-learn

## Notes

These are learning exercises, not standalone projects — kept here to document the fundamentals behind the more focused projects in my other repositories (DNA sequence analysis, biomedical signal processing, AI projects).
